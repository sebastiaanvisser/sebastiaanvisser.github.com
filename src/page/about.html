About me

Hi, I'm Sebastiaan Visser. I'm one of the engineers behind Silk, a tool that brings the semantic web to the masses. In my spare time I'm involved in some open source software projects, mainly written in Haskell and JavaScript. Besides software I'm interested in design, typography, computer graphics, cycling, and running.

I live in Utrecht and work in Amsterdam, beautiful cities that I both very much enjoy. If you want to contact me, you can do so by e-mail (my first name at this domain) or twitter (@sfvisser).
